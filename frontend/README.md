## Getting Started

### Prerequisites

Before you begin, ensure you have met the following requirements:
- Node.js (v12 or higher recommended)
- npm (v6 or higher) or Yarn (v1.22 or higher)

1. Install the dependencies

```
npm install
```

or 

```
yarn install
```

## Running the App

```
npm run dev
```
